# Strategy

Koer `/elle-strategy` skill med det angivne klient-navn.

## Faser
1. **Context** — spoergsmaal + eksisterende klientdata
2. **Research** — NotebookLM + web research + performance-data
3. **Planning** — positionering, maalgruppe (5H), funnel, kanal-plan
4. **Execution** — skriv komplet strategidokument
5. **Review** — godkendelse, feedback-loop, afledte outputs

Checkpoint mellem hver fase — brugeren godkender foer naeste starter.

Brug argument som klient-navn. Eksempel: `/strategy zizzi`
